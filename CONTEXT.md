# 프로젝트 컨텍스트

## 작업 배경
AI 반도체 투자 전략 PDF(Gap and Go, VCP, 3:3:4 피라미딩)를 분석하고,
해당 전략의 통계적 유효성을 검증하기 위해 파이썬 백테스팅 시스템을 구축 중.

## 개발 환경
- Windows 11, Python 3.10, VS Code
- 노트북 ↔ 데스크탑 번갈아 작업 (GitHub으로 동기화)
- 한국 주식 버전은 추후 KIS Open API 연동 예정

---

## 실험 로그 (누적)

### [실험 0] Gap and Go 전략 검증 (초기 검증)
- NVDA 2년, PLTR 3년 대상
- 갭 3% + 거래량 2배 동시 충족 → NVDA 0건, PLTR 17건 (당일 평균 -0.22%)
- **결론:** PDF 전략은 퀀트 전략이 아닌 패턴 매매. 조건값에 통계적 근거 없음

---

### [실험 1] 청산 방식 비교 (기준 실험)
**조건:** 슈퍼사이클 19종목, 5년, $10,000, bear_filter=none, stop_mode=pct12

| 청산 방식 (exit_mode) | 수익률 | CAGR | MDD | 샤프 | 거래수 |
|----------------------|--------|------|-----|------|--------|
| current (단계적 MA) | +150.0% | 24.6% | -15.9% | 1.16 | 214건 |
| fast (MA10 즉시전량) | +169.0% | 26.8% | -13.6% | 1.24 | 130건 |
| confirm (MA20 3일확인) | +173.7% | 27.4% | -16.9% | 1.22 | 198건 |
| **hybrid (MA10→50% + MA20확인→잔여전량)** | **+188.4%** | **29.0%** | **-16.8%** | **1.27** | **137건** |

SPY Buy&Hold: +82.9%  
**채택:** hybrid (수익률·CAGR·샤프 모두 최우수)

---

### [실험 2] Bear Filter 비교 (구버전, AI 반도체 27종목 기준)
- 필터 없음: +93.5%
- block (SPY 200일선 아래 진입 금지): +99.4% ← 당시 채택
- strict (Strong만 허용): +99.1%
- **주의:** 이 실험은 현재 19종목 유니버스·코드와 다른 구버전 기준. 재실험 필요

- stop_mode: pct12
