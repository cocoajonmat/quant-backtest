# /done — 실험 마무리 & 커밋 푸시

세션에서 진행한 실험을 마무리할 때 실행한다.

## 순서

1. **CONTEXT 파일 점검 및 업데이트**
   - `CONTEXT.md` — 인덱스의 현황/최고기록/다음실험이 최신인지 확인 후 수정
   - `CONTEXT_trend.md` — 진행한 실험이 로그에 누락 없이 기록됐는지 확인
   - `CONTEXT_supercycle.md` — 슈퍼사이클 실험을 진행했다면 동일하게 확인
   - `CLAUDE.md` — 프로젝트 현황 표와 채택 파라미터가 최신인지 확인 후 수정

2. **git 상태 확인**
   - `git status`, `git diff` 로 변경 파일 목록 파악

3. **커밋**
   - 변경된 파일만 스테이징 (`.py`, `.md` 위주)
   - 커밋 메시지: `feat: <오늘 진행한 실험 시리즈 요약>` 형식
   - 예시: `feat: 일반 추세추종 K1~K2 — linreg 윈도우 90일 확정, heat_cap 10% 채택`

4. **푸시**
   - `git push` 로 원격 반영

## 주의
- 백테스트 결과 이미지(`backtest_result.png`) 등 바이너리는 커밋에서 제외
- CONTEXT 파일에 실험 결과가 빠져 있으면 먼저 보완하고 커밋
