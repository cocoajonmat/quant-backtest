## 세계적 추세추종 트레이더와의 비교 분석

| 트레이더/시스템 | 스타일 | 대표 지표 |
|---------------|--------|---------|
| **터틀 트레이더스** (Richard Dennis) | 채널 돌파, 고정 ATR 손절, 20~50개 시장 | 승률 35~40%, 손익비 3~5x |
| **AQR / Man AHL** (퀀트 CTA) | 시계열 모멘텀, 다자산, 12개월 신호 | 샤프 0.5~0.8 (실전 기준) |
| **Andreas Clenow** (Stocks on the Move) | 52주 채널, 연간 모멘텀, 포지션 사이징 | 승률 40~45% |
| **Gary Antonacci** (듀얼 모멘텀) | 절대+상대 모멘텀, 월간 리밸런싱 | CAGR 14~18%, MDD -20~35% |
| **Mark Minervini** (SEPA) | 52주 신고가 + VCP + 펀더멘털 | 승률 50%+ (스윙) |

### Clenow 지수회귀 방식 (H 시리즈에서 채택)
```python
log_prices = np.log(close[-90:])
slope, _, r_value, _, _ = scipy.stats.linregress(range(90), log_prices)
momentum_score = slope * 252 * (r_value ** 2)   # 연환산 기울기 × 추세 선명도
# 최소 게이트: score > 0.15
```
기울기(얼마나 빠르게) × R²(얼마나 일정하게) → 변동성 크고 들쑥날쑥한 종목은 자동 탈락.
