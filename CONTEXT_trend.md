# 일반 추세추종 전략

## 전략 개요
광범위한 유니버스(NDX100)에서 "지금 슈퍼사이클 초입 특성을 보이는 종목"을 실시간 감지해
기존 슈퍼사이클 전략(hybrid/ATR/trailing_stop)과 동일한 방식으로 진입하는 전략.

**핵심 철학 (방향 A: 동적 슈퍼사이클 감지)**
- 수작업 선별 대신 실시간 모멘텀 가속 필터로 슈퍼사이클 초입 종목 동적 포착
- 유니버스 품질이 섞이는 만큼 bear_filter를 더 타이트하게 운용
- 슈퍼사이클과 동일한 진입/청산 메커니즘 유지 (검증된 파라미터 재활용)

**방향 B (미구현): 광범위 추세추종 (터틀 스타일)**
- ATR 1~2%, 타이트 손절, max_positions 20~30
- 승률 30~35%여도 손익비로 수익 내는 구조
- 방향 A 결과 충분히 나온 후 착수 예정

### 핵심 개선사항 — 모멘텀 측정 방식 (H 시리즈에서 완료)

**기존 방식의 문제:**
- `ret_3m > ret_12m / 4` 조건은 "3개월에 7~10% 오르면 통과" — 반등 종목도 다 들어옴
- "얼마나 올랐나(크기)"만 측정, 추세의 선형성(노이즈 여부)은 측정 불가

**Clenow 지수회귀 방식 (채택):**
```python
log_prices = np.log(close[-90:])
slope, _, r, _, _ = scipy.stats.linregress(range(90), log_prices)
momentum_score = slope * 252 * (r ** 2)   # 연환산 기울기 × 추세 선명도
# 최소 게이트: score > 0.15
```
- 기울기(얼마나 빠르게) × R²(얼마나 일정하게) → 변동성 크고 들쑥날쑥한 종목은 자동 탈락
- 8년 기준 샤프 0.90 → 1.14, 수익률 +469% → +831%

### 아직 적용 안 한 세계 표준 개선사항 (우선순위순)

1. **포트폴리오 열 리스크 캡** — 진입 전 포트폴리오 총 리스크 합산, 12% 초과 시 진입 스킵
2. **진입 신호 단순화** — 5팩터 점수합산 → "52주 신고가 5% 이내 + MA 정배열 + ADX 20+" AND 조건
3. **청산 로직 단순화** — 현재 7개 규칙 → 하드스탑/트레일링/MA20 단일 3개로 축소
4. **상관관계 집중도 제한** — 동일 섹터 포지션 합산 리스크 캡

## 파일

`trend_backtest.py` — 방향 A 백테스트 메인 파일

**동적 유니버스 선정 기준 (`get_dynamic_universe`, 현재 채택 방식)**
1. 12개월 수익률 > ret12_min (현재 20%) — 완전 하락 종목 안전망 필터
2. 6개월 수익률 > 0 — 추세 지속성 확인
3. **지수회귀 기울기×R² > 0.15** (linreg gate) — 90일 추세 강도+선형성 (Clenow 방식)
4. 현재가 > MA50 > MA200 (정배열)
5. ADX >= 20 (추세 강도)
6. 일평균 거래대금 $100M 이상 (유동성)
→ linreg 점수 내림차순 Top 5 선택
