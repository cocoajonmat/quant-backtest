# 일반 추세추종 전략

## 전략 개요
광범위한 유니버스(NDX100)에서 "지금 슈퍼사이클 초입 특성을 보이는 종목"을 실시간 감지해
기존 슈퍼사이클 전략(hybrid/ATR/trailing_stop)과 동일한 방식으로 진입하는 전략.

**핵심 철학 (방향 A: 동적 슈퍼사이클 감지)**
- 수작업 선별 대신 실시간 모멘텀 가속 필터로 슈퍼사이클 초입 종목 동적 포착
- 유니버스 품질이 섞이는 만큼 bear_filter를 더 타이트하게 운용
- 슈퍼사이클과 동일한 진입/청산 메커니즘 유지 (검증된 파라미터 재활용)

**방향 B (미구현): 광범위 추세추종 (터틀 스타일)**
- ATR 1~2%, 타이트 손절, max_positions 20~30
- 승률 30~35%여도 손익비로 수익 내는 구조
- 방향 A 결과 충분히 나온 후 착수 예정

---

## 파일

`trend_backtest.py` — 방향 A 백테스트 메인 파일

**동적 유니버스 선정 기준 (`get_dynamic_universe`)**
1. 12개월 수익률 > ret12_min (기본 40%)
2. 3개월 수익률 > 12개월 수익률 / 4 (모멘텀 가속)
3. 현재가 > MA50 > MA200 (정배열)
4. ADX >= adx_min (추세 강도)
5. 일평균 거래대금 $100M 이상 (유동성)
→ 3개월 수익률 내림차순 Top top_n 선택

## 현재 채택 파라미터 (실험B3 기준)
- 유니버스: NDX100 동적 감지 / top_n=8 / ret12>40% / ADX>=20
- bear_filter: block MA100 ← 슈퍼사이클과 다름 (MA200보다 효과적)
- exit_mode: hybrid
- stop_mode: pct12
- trailing_stop: original
- atr_sizing: on, atr_risk_pct: 4.0%, cap: 40%
- max_positions: 4
- adx_threshold: 20
- min_hold_days: 3

**현재 최고 기록 (2026-05-08 기준)**
수익률 +240.2% / CAGR 34.2% / MDD -21.5% / 샤프 1.13 / SPY +86.9%

**결론:**
- top8 / ret12>40%가 최적 (A1)

- NDX100 실험12(샤프 0.83) 대비 A1(샤프 1.08)로 유의미한 개선
